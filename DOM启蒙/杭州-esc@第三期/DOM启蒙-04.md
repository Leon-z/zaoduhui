# 《Dom启蒙》读书笔记

## 第5章 元素节点几何量与滚动几何量

- 5.1 元素节点尺寸、偏移及滚动概览
- 5.2 获取元素相对于 offsetParent 的 offsetTop 及 offsetLeft 值
- 5.3 使用 getBoundingClientRect() 获取元素相对于视区的Top, Right, Bottom及Left边沿偏移量
